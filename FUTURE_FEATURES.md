# Future Features

This document outlines planned features for OC Message Explorer. Items are listed in priority order.

## High Priority

### Tag-based Filtering
- Add tag cloud widget in sidebar
- Filter messages by clicking on tags
- Show tag count in cloud
- Support for tag combinations (AND/OR logic)

### OpenAI Integration
- Add settings modal for API key configuration
- List models from OpenAI API endpoint
- Allow model selection for prompt optimization
- Implement prompt rewriting using selected model
- Save API key configuration securely

## Medium Priority

### Enhanced Search
- Add advanced search options (date range, tags, type)
- Save search queries as presets
- Search within selected messages only
- Regex support in search queries

### Message Organization
- Add message archiving
- Create message templates
- Bulk tag assignment
- Message duplication with auto-renaming

### Export Formats
- Export as PDF
- Export as Markdown
- Export as plain text with formatting
- Export individual messages or folders

## Low Priority

### Customization
- Theme selection (light/dark/custom colors)
- Font size adjustments
- UI layout customization
- Keyboard shortcuts configuration

### Advanced Features
- Message versioning
- Compare message versions
- Merge similar messages
- Auto-organize messages by similarity

## Considered but Deprioritized

### Real-time Collaboration
- Multiple users viewing same messages
- Shared message collections
- Comments and annotations
- Sharing links to specific messages

### Mobile App
- Native iOS and Android apps
- Mobile-responsive web improvements
- Offline mode

---

**Note**: This list is subject to change based on user feedback and technical constraints.
